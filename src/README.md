# GUIDELINES (STRICT) 
## Project Structure
Stretch
-	/assets → images, icons, UI files
-	/docs → documentation
-	main.cs → main program
-	/models → data classes (e.g., TaskModel.cs)
-	/repositories → database operations (e.g., TaskRepository.cs)
-	/controllers → handle app logic (e.g., TaskController.cs)
-	/services → reusable services/methods (e.g., TaskService.cs)
-	/views → UI elements (forms, dashboards)
-	README.md → instructions for this part
- /database
  - DBConnection.cs → database connection
- README.md → project overview
